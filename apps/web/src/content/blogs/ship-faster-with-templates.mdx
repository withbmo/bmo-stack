# Reusable runtime templates

Templates reduce setup time and improve consistency across teams.

## What to include

- Health checks and baseline monitoring.
- Standard environment variable handling.
- CI/CD defaults for test and deploy.

## Team adoption

- Keep templates versioned.
- Document intended usage clearly.
- Gather feedback every release cycle.

## Final note

Good templates improve velocity without removing ownership from developers.
